# TraderCRM
برنامج اداره عملاء و دورات مزارع 
اول حاجه عندنا المزارع 
بنعرف موديول المزارع ك الاتي 
-----------------------------------------------------------------
@Entity (tableName :"Farmtbl") بنخزن فيه كل بيانات المزرعه 
اول حاجه طبعا ال 
1-id : string autogenrate
وده مفتاح رئيسي

2-farm name  اسم المزرعه 
3-farmNote   ملاحظات
4-blocked حاله المزرعه تقبل الشغل ولا محظورة مينفعش اربط عليها حاجه
------------------------------------------------------------------
الدورات 
@ُEntity (tableName : "Cycletbl")
1-id : string autogenrate  الخاص بالدورة ك مفتاح رئيسي
2-Farmid وده المفتاح بتاع المزرعه
طبعا لكل مزرعه عدة دورات / لكل دورة مزرعه واحدة
3-cycleName اسم الدورة 
4-sd : shortdate تاريخ البداية 
5-ed :shortdate تاريخ النهاية
6-isActive  "نشط و غير نشط"  نشطه ولا لاء؟
